Breen Team Arcade API

Server for the game website in which you can access data by endpoints in below

Nav for endpoints

  • Users
  • Scores for Orc of the Rings

Endpoints

  • GET

    • /api/users - a list of users
    • /api/users/:userid - a user by user id
    • /api/games/:gamesid/scores - paginated scoress by game

  • POST

    • /api/users - create a user
    • /api/games/:gameid/scores - create a score

  • DELETE

    • /api/users/:userid - create a user
