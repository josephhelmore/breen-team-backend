Breen Team Arcade API

This API homepage will provide some basic information about the Breen Team Arcade Server for the game website. To view the live version of our app, you can click here.

The Arcade is designed with a library of games in mind, providing the user with a handful of options of which games to choose from. So far we have developed a runner game, 'Orc Of The Rings', a minesweeper style game, 'Debugger' and a card 'Memory Game'.

  • Scores for Orc of the Rings

Endpoints

GET requests

  • /api/users - recieves a list of users
  • /api/users/:user_id - recieves information about a specific user via user id
  • /api/games/:games_id/scores - recieves a paginated list of scores via game_id

POST requests

  • /api/users - This allows the FE to create a user
  • /api/games/:gameid/scores - This allows the FE to add a score to the specified game
  • /api/games/:game_id - When needed, this allows the team to add a new game

DELETE

  • /api/users/:userid - Allows the FE to delete a user when needed
  • /api/games/:game_id - When needed, this allows the team to delete a game if needed
